# 高TPACS分子设计：引入空间几何特征

本仓库包含我们关于双光子吸收截面（TPACS）预测研究的代码与数据。我们在先前发表的6特征模型基础上，引入了**空间几何描述符**，以捕捉传统研究中忽略的三维信息对TPACS的影响。

---

## 📖 概述

双光子吸收材料在生物成像、光动力治疗及光限幅等领域具有重要应用。此前研究表明**共轭长度**是主导因子，但无法完全解释TPACS的差异。我们假设分子**形状、尺寸及刚性**等几何特征携带额外的物理信息。

在本工作中，我们：
- 从856个有机发色团（实验TPACS，波长600–1100 nm）出发；
- 构造三类空间几何特征：**整体形状**（回旋半径`rg`、非球形度、离心率）、**立体异构**（手性中心数、E/Z双键数）及**分子刚性**；
- 将`rg`纳入原始6特征XGBoost模型，并通过贝叶斯搜索（Optuna‑TPE）重新优化超参数；
- 获得一致的性能提升（R² 0.587 → **0.601**），且`rg`在SHAP重要性中排名第三，验证了其物理相关性。

---

## 📝 依赖库

- Python ≥ 3.8
- RDKit ≥ 2022.03
- XGBoost ≥ 1.6
- Optuna ≥ 3.0
- NumPy, pandas, matplotlib, seaborn
- SHAP ≥ 0.41
